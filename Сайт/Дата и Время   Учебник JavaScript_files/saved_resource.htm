                  Там в массиве d к каждому элементу даты, кроме года (месяц, число, минуты, часы) прибавляется строка '0' спереди.
                  Допустим у нас есть дата: 5-11-2014 4:45 соответственно у нас получится такой массив d: ['05', '011', '2014', '04', '045']
                  Дальше идет цикл, в котором берется каждая строка из массива и обрезается до последних 2 символов. Метод slice(-2) скопирует последние 2 символа строки и эти 2 символа присваиваются текущему элементу массива. Таким образом от каждой строки в массиве останется последние 2 символа и получится такой массив: ['05', '11', '14', '04', '45'] И получается, что там где был 0 лишний, он убирается. Ну а дальше первые 3 элемента массива собираются в строку через разделитель '.' и получается строка '05.11.14', а оставшиеся элементы собираются в строку через разделитель ':' и получается строка '04:45'. Ну и эти строки складываются через пробел и получается строка '05.11.14 04:45', которую и возвращает функция.

                      function formatDate(date) {

                      var dd = date.getDate()
                      if ( dd < 10 ) dd = '0' + dd;

                      var mm = date.getMonth()+1
                      if ( mm < 10 ) mm = '0' + mm;

                      var yy = date.getFullYear() % 100;
                      if ( yy < 10 ) yy = '0' + yy;

                      return dd+'.'+mm+'.'+yy;
                      }

                                                                                              Так где написано, что объект даты можно так создавать? Где написано, что он может корректно обработать строку вида "yyyy, mm, dd"? Если уж создаете объект даты через строку, то используйте строку формата "yyyy-mm-dd". Причем в этой строке месяц начинается как обычно с 1. А год, месяц и день должны иметь столько символов сколько указано в шаблоне. Т.к. для 5 января 2012 года строка должна иметь вид: "2012-01-05", а не "2012-1-5".
                                                                                              А если хотите создавать объект даты через несколько параметров, то не передавайте туда одну строку а предавайте отдельно год, месяц и день, например так:

                                                                                              var year = +prompt('введите год xxxx','');
var month = +prompt('введите месяц xx','') - 1;
var day = +prompt('введите день xx','');
var date = new Date(year, month, day);
alert(date);
